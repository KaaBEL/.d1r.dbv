This directory contains the two directories developed within MS1 source code for v.0.10.4.8 `Modular spaceships\Assets\ConsoleTest` and `Modular spaceships\Assets\Localhost Mod`.

# Console App Test
Localhost Mod contains VS2022 solution, which can be used to test via console app interface, without the need of unity editor. Projects should be launchable after opening `Localhost Mod.sln` in Visual Studio. You should be able get the responses at http://localhost:5501/Ships/ , http://localhost:5501/Ships/defaultship_destroyer.mssss?param=value or http://localhost:5501/Ships/defaultship_fighter.png#hash , but it may require creating setting.txt in the same directory as produced executable, containing a path to MS1 ships folder such as: `C:\Users\<user>\AppData\LocalLow\Skyscraper Labs\Modular spaceships`.
